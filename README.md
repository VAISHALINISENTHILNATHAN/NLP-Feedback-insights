# NLP Feedback Insights

Turn user feedback into actionable insights using Natural Language Processing (NLP).  
Analyze sentiment, extract topics, and gain meaningful insights to improve products, services, and customer experiences.

## Features

- **Sentiment Analysis**: Automatically detect positive, negative, or neutral sentiment in feedback.
- **Topic Extraction**: Identify key themes and topics from user comments.
- **Actionable Insights**: Summarize feedback to highlight trends and areas for improvement.
- **Visualization**: Optional charts and graphs for better understanding of feedback patterns.

## Running

- create review.csv file
- run the ipynb files in the order (preprocessing, topic modeling, sentiment, insights, app)
- run the final_run file
- open anaconda prompt and run streamlit version
